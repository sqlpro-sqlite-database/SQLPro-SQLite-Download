#### System Compatibility
- **Supported macOS versions:** 10.14 (Mojave) and newer
- **Architecture support:** Intel and Apple Silicon (Universal Binary)
- **Disk space required:** Minimum 150MB for application, additional space for databases

#### Installation Methods

**Direct Installation:**
1. Download the application package
2. Open the downloaded .dmg file
3. Drag SQLPro for SQLite to your Applications folder
4. Launch from Applications or Launchpad

**First-Time Setup:**
1. On first launch, grant necessary permissions if prompted
2. The application will create default settings and workspace
3. Begin by opening an existing SQLite file or creating a new database
